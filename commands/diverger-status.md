# /diverger-status

Muestra el stack tecnológico detectado y el estado actual de la configuración.

## Uso
```
/diverger-status
```

## Comportamiento
1. Muestra la versión de diverger-claude y fecha de última generación
2. Lista los profiles aplicados
3. Re-detecta el stack y muestra las tecnologías con nivel de confianza
4. Muestra información de monorepo si aplica
5. Muestra la arquitectura detectada
